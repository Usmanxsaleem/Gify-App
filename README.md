⚙️ How It Works

User enters a search keyword.

JavaScript captures the input from the search field.

A request is sent to the Giphy API.

The API returns GIF data in JSON format.

JavaScript dynamically renders the GIFs on the page.

This project uses the Giphy Developer API.

You can get your API key here:

https://developers.giphy.com/

Example API endpoint used:

https://api.giphy.com/v1/gifs/search?api_key=YOUR_API_KEY&q=SEARCH_TERM&limit=12
